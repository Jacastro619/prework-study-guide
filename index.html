HTML

  • Enter your HTML notes here
  • The "head" element contains information about the webpage.
  • The "body" element represents the visible content shown to the user
  • The "h" element is for header and it is followed by a number to determine the tier of the specific heading
  • The "ul" element means unordered list and it is followed by a "li" element that is know as the listed item

CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class is used to target a specific element that is given a class to change only that specific element with the matching class. This can be done while not changing the same element/s that do not have the matching class

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push allows us to push the current version into the current branch

JavaScript

  • Enter your JavaScript notes here
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Console.log logs something to the developers console
  • For loop is used to interact with all elements in an array
  • An array is zero indexed meaning the first index in an array is 0
